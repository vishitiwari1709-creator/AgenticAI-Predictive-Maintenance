# Vehicle Safety & Maintenance Prototype

A web-based prototype application for vehicle safety and maintenance with role-based dashboards and AI voice assistant.

## Features

- **Role-based Access**: Three distinct user roles (Owner, Technician, Manager)
- **Interactive Dashboards**: Role-specific information and actions
- **AI Voice Assistant**: Text-to-speech and speech-to-text capabilities
- **Responsive Design**: Works on desktop and mobile devices
- **Safety & Maintenance Tools**: Fault reporting, service booking, compliance tracking

## User Roles

### Vehicle Owner
- View vehicle safety status
- Report faults and book services
- Access safety briefings and recall information

### Technician
- Access virtual SOPs and technical documentation
- View assigned service requests
- Track certification requirements

### Fleet Manager
- Monitor KPIs and compliance metrics
- Oversee technician training status
- Dispatch service requests

## Getting Started

1. Clone or download this repository
2. Open `index.html` in a modern web browser
3. Use the following credentials to login:
   - Email: `owner@example.com`, `technician@company.com`, or `manager@company.com`
   - Password: `password`

## Technology Stack

- HTML5, CSS3, JavaScript (ES6+)
- Web Speech API for voice interactions
- CSS Grid and Flexbox for layouts
- Responsive design principles

## Browser Compatibility

- Chrome (recommended for best speech recognition)
- Firefox
- Safari
- Edge
